"# billetera"  

Descripción de la actividad: 
Desarrollo de una aplicación web para la gestión de finanzas personales, permitiendo realizar depósitos, transferencias a contactos y visualización de historial.

Desafío principal: 
Implementar una lógica de transacciones robusta que actualizara el saldo en tiempo real y persistiera los datos localmente sin perder la integridad de la información.

Solución propuesta: 
Se utilizó un sistema de validaciones cruzadas para evitar saldos negativos y se estructuró la información en LocalStorage, permitiendo una experiencia de usuario fluida y sin pérdida de datos al recargar.

Herramientas técnicas: 
HTML5 semántico, SASS (arquitectura modular), JavaScript (ES6+) y jQuery.

Principales aprendizajes: 
Manejo avanzado del DOM, gestión de eventos asíncronos y aplicación de patrones de diseño para organizar el código CSS.

Métricas de impacto:
Reducción del tiempo de carga mediante la optimización de selectores CSS.
100% de efectividad en la persistencia de datos durante las pruebas de usuario.

Habilidades técnicas aplicadas: 
Programación funcional básica, manipulación de objetos JSON, diseño responsivo con Bootstrap 4 y metodologías ágiles.

Justificación:
Este proyecto demuestra mi capacidad para resolver problemas lógicos de negocio (manejo de dinero/datos) aplicados a una interfaz moderna y funcional.
